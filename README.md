# AI Resume Analyzer & Job Matcher

An AI-powered web application built with Python that helps job seekers analyze their resumes, 
identify key skills, match them with job descriptions, and discover skill gaps.

This project uses Natural Language Processing (NLP) and Machine Learning concepts to provide:
- Resume skill extraction  
- Job description matching  
- Match percentage scoring  
- Skill gap analysis  
- Improvement suggestions  

Tech Stack: Python, Streamlit, Pandas, spaCy, scikit-learn, PyPDF2
